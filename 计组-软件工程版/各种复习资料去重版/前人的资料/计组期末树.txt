第三章 计算机功能和互联的顶层视图
    计算机功能和连接
        顶层来看计算机有的功能
            指令的取址和执行
            中断
            IO
        顶层来看计算机的连接
            总线概念
                总线种类：数据总线（方向：CPU->各个总线），地址总线（双向的），控制总线（双向的）
            总线仲裁
                数据总线地址总线一般要抢，控制一般不用抢
                方法
                    1.集中式（搞懂搞透）
                        菊花链式（串式链接）
                            特点：典型的一个请求线连起所有设备。前面设备响应，
                                  后面不会响应，离控制部件越近优先级越高，若是前面设备与控制部件的连接				  断掉，后面会连不上
                            优点：有优先级
                            缺点：不公平
                        计数机制（轮询机制）
                            实现方法：时间取模机制
                            优点：公平
                            缺点：慢
                        集中控制机制
                            特点：有一个判定逻辑，判定谁优先
                    2.分布式
            时序
                同步异步概念
                    同步：大时钟控制
                    异步：个体事件的先后关系，一件事情激发另外一件事情
                        第一周期：出地址
                        第二周期：发出读命令
                        第三周期：出数据
                PCI不考，实例不考

    存储体系结构
        1.内外有别 寄存器 -> Cache -> 内存 -> 硬盘 -> 光盘 -> 磁带
        概念：internal memory , main memory
    访问模式
        寄存器以字word为单位
        寄存器以块block为单位
        内存访存以word为单位
        磁盘以扇区block为单位
    访问方法
        随机访问模式 内存，寄存器
            特点
                1.按地址
                2.所有单元时间都相等
        直接访问方式 磁盘，光盘
            特点
                1.按地址
                2.单元访问时间不同
        顺序访问方式 磁带
            特点
                1.不按地址，从头读到需要寻找的地方
